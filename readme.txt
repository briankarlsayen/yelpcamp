# Yelpcamp

This project was created using Node.js, Express, MongoDB, and Bootstrap. Passport.js was used to handle authentication.

## Start the project

1. Clone the project
2. Install the packages
```
npm install https://github.com/briankarlsayen/yelpcamp.git
```
3. Create mongoDB
https://www.mongodb.com/
inside collections create a Database
4. Add environment variables
add new file name .env
inside .env file
```
mongodb+srv://<username>:<password>@cluster0.eygh2.mongodb.net/<databasse-name>?retryWrites=true&w=majority
```
note: replace <> files with your own.
5. Run the project
```
npm run start
```
